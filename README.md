## Getting Started

### Prerequisites

- Node.js (version 14.x or higher)
- npm or Yarn

### Installation

1. Clone the repository
2. intall depedencies using `npm i`

### How to Run
- local env
1. `npnm run dev` or `yarn dev`

- Prod env
1. `npm run build`
2. `npm start`
